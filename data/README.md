# data/

Place licensed EEG datasets here locally. **Do not commit dataset binaries.**

| Folder | Contents |
|--------|----------|
| `raw/` | DEAP preprocessed subject files (`s01.dat` … `s32.dat`) |
| `seed/` | SEED NPZ files expected by the loader (`DatasetCaricatoNoImage.npz`, `LabelsNoImage.npz`, `SubjectsNoImage.npz`) |

Obtain datasets from their official sources. See the root `README.md` (Dataset Setup) for details.
